Android setup - GPS Logger for Android

This app works nicely but requires a few steps to setup properly in order to track the race.

First after you install the app, click on the top left button to access the settings:

Once in the settings we'll go in each section to set a few parameters:

  • Logging details - Log To GPX
    Swipe to the right to activate logging to GPX
  • Logging details - Log to KML, Log to Custom URL, Log to OpenGTS Server, Log to Plain Text, Log to NMEA
    Swipe to the left to disable logging in these file formats.
  • Logging details - New File Create
    I recommend setting it to 'Every time I start'
  • Performance - Location Providers
    Set to GPS, disabling network is better because on the water Network positions are going to be very bad anyway and ignored on the website.
  • Performance - Time Before Logging
    I set it to 2 seconds, this is the interval at which the GPS chip in your phone will be activated and wait for a fix. It'll probably sampled down to 5 sec interval on the website, but better log a little more than too few data points.
  • Performance - Keep GPS on between fixes
    I recomend to set it ON, the positions are going to be much better. Otheriwse it'll turn the GPS on and off all the time,
    which will lengthen the time the GPS chip needs to aquire a fix on your position.
  • Performance - Distance Filter
    Set it to 1 meter, this is going to be the minimal resolution of your data
  • Performance - Accuracy Filter
    I set it to 5 meters. This is the expected GPS precision before a point is recorded.
    If your phone does not log any positions or seems slow, enter a higher value such as 8, then 10.
    By entering a higher value, you leave more room for error on your GPS fix.

Logging

It is best to start logging like 30 min ahead of the start, and stop as soon as possible after the race.

The smaller the file size the faster it can be added to the website, if you log for the whole day, I'll have to find beginning and ends of the race among your log which takes some time.

Sharing your data

Once you are done logging, you can share your data by pressing the bottom right button:

You will be prompted to pick a file, chose the latest one, which hopefully corresponds to your race data.

After you hit OK, another tab will show up offering you options to share. Chose to share via email.

Send the email to chicago.beercanracing@gmail.com

I should then receive your data and put it online in a matter of an hour or so after we are back to our dock and I have my rum and coke!
